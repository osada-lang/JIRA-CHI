ホーム / 実績紹介 / ミリオン・タイマー / プライバシーポリシー

このプライバシーポリシー（以下「本ポリシー」）は、「ミリオン・タイマー」（以下「本アプリ」）の提供にあたり、利用者の情報の取り扱いについて説明するものです。

1. 収集する情報と利用目的

本アプリは、以下の情報を収集し、利用目的に応じて処理します。

  • Google ログイン情報: Google アカウントの ID、メールアドレス、プロフィール画像（ユーザー認証と本人確認の目的）
  • ポイント情報: 獲得ポイント数、交換履歴、交換額（ポイント管理と交換処理の目的）
  • タイマー利用情報: タイマー使用回数、広告視聴回数、タイマー時間（サービス改善と不正利用防止の目的）
  • デバイス情報: デバイス ID、OS バージョン、アプリバージョン（サービス提供と不正利用防止の目的）
  • 位置情報: 収集しません

2. Google ログイン（Firebase Authentication）

本アプリは、Google ログインを使用してユーザー認証を行います。Google ログイン時に、Google から提供される ID、メールアドレス、プロフィール画像を取得します。これらの情報は Firebase Authentication で管理され、本人確認とアカウント管理の目的でのみ使用されます。Google のプライバシーポリシーについては、Google プライバシーポリシーをご参照ください。

3. ポイント管理（Firestore）

獲得したポイントは、Google Cloud の Firestore（NoSQL データベース）で一元管理されます。ポイント情報は暗号化されて保存され、本人確認済みのユーザーのみがアクセスできます。ポイント二重計上の防止、複数デバイス間での同期、および不正利用防止の目的で使用されます。

4. 広告とリワード検証（AdMob）

本アプリは Google AdMob を使用してリワード広告を配信しています。AdMob は、広告の視聴を検証し、リワード付与の正当性を確認します。AdMob は、デバイス ID、広告視聴時刻、視聴完了情報などを収集します。詳細は Google 広告に関するポリシーをご参照ください。

5. AdMob Server-to-Server Verification

リワード広告の視聴完了を検証するため、AdMob Server-to-Server Verification を使用しています。これにより、Google 側でリワード付与の正当性を確認し、開発者ツールでのポイント改ざんを防止しています。

6. ポイント交換（えらべるPAY API）

ポイントを景品に交換する際、本アプリは Giftee の「えらべるPAY」API と連携します。交換時に、ユーザーの ID、交換ポイント数、交換額などの情報が Giftee に送信されます。Giftee のプライバシーポリシーについては、Giftee 公式サイトをご参照ください。

7. 通信セキュリティ

本アプリと Google Cloud（Firebase）間の通信は、HTTPS により暗号化されています。Firebase Security Rules により、認証されたユーザーのみが自身のデータにアクセスできるよう制御されています。

8. API キー管理

えらべるPAY API キーは、環境変数（`.env`）で管理され、Git リポジトリには含まれません。API キーは Firebase Cloud Functions 内でのみ使用され、クライアント側には公開されません。

9. 上限管理とアラート

ポイント交換は、以下の上限で管理されています。

  • ユーザーごとの上限: 1日1万円
  • 全体の上限: 1日10万円
  • アラート: 1日の交換合計額が 50,000円 を超えた場合、管理者にメール通知

これらの上限は、不正利用による被害を最小化し、ユーザーの安全を確保するために設定されています。

10. 情報の保持期間

ユーザーがアカウントを削除した場合、関連するすべての個人情報は Firestore から削除されます。ただし、法律で定められた期間は、適切に保管される場合があります。

11. 第三者への情報提供

本アプリは、ユーザーの同意なしに個人情報を第三者に提供することはありません。ただし、以下の場合は例外とします。

  • 法律で定められた場合
  • ユーザーの安全や権利を保護するために必要な場合
  • ポイント交換時に Giftee へ必要な情報を提供する場合

12. ユーザーの権利

ユーザーは、以下の権利を有しています。

  • 自身の個人情報の開示を請求する権利
  • 自身の個人情報の修正を請求する権利
  • 自身の個人情報の削除を請求する権利（アカウント削除）
  • 個人情報の処理に異議を唱える権利

これらの請求は、お問い合わせフォームからご連絡ください。

13. ポリシーの変更

本ポリシーは、予告なく変更される場合があります。変更があった場合は、本アプリ内またはウェブサイトで通知されます。

14. お問い合わせ

本ポリシーに関するご質問やご不明な点がございましたら、お気軽にお問い合わせください。
